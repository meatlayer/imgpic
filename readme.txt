Скрипт — хостинг-изображений "Imgpic" v 1.0
Быстрый, лёгкий, безопасный и без сложностей в установке.

- загрузка изображений
- выбора качества на выходе
- выбор размера на выходе
- сохранение пропорций
- выдача готовых ссылок
- обратная связь
- страницы
- аватарки :)

УСТАНОВКА:
1. Распаковываем архив со скриптом в корень вашей веб-директории (на которую настроен домен сайта)
2. Открываем файл config.php и изменяем "абсолютный путь к папке с сайтом", "домен сайта", "почтовый ящик для обратной связи".
3. Готово.
